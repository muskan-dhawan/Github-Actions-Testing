# GitHub Actions Testing

A project for learning and testing GitHub Actions workflows, CI/CD pipelines, and automation using GitHub.

## 🚀 Features

* GitHub Actions workflow setup
* CI/CD pipeline testing
* Automated workflow execution
* GitHub integration practice

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/muskan-dhawan/Github-Actions-Testing.git
```

Move into the project directory:

```bash
cd Github-Actions-Testing
```

Install dependencies:

```bash
pip install -r requirements.txt
```
